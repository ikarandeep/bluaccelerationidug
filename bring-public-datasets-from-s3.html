Bring public datasets from S3

Kelly the data analyst at Public Policy Institute want to be able to load publicly available data sets directly from AWS S3 object storage to BLU Acceleration for cloud and use built-in analytic feature to analyze data.

1) Login with the provided user name/password

2) Click Manage -> Work with Tables from the main menu

3) Click + button -> DDL statement editor will open on the right

4) Download the DDL file here: censusPopulation_new.ddl.txt

5) Copy-Paste DDL statements for the tables that you want to create - Click Run DDL button
Users can also create all of the tables from the attached file at once

6) Click Manage -> Load Data from the main menu

7) Open "Schedule a load" section

8) Choose Amazon S3 -> Browse public bucket -> Provide bucket name, e.g. ibm.idug.public -> Browse S3 Files

9) Select the data file that you want to load as below -> OK
Try these data files:
Ancestry.csv can be selected for loading data in to Ancestry table
ss12pca.csv or person_2_5gb.del can be selected for loading data in to Person table

10) Separator character and date/time options can be set as below -> Next

11) Select Schema which will be same as your user name as in step 1, also select the table that you are interested in loading

12) Choose either of the options below at "Select a table" step

13) Select date and time when you want to run this load job -> Next -> Finish -> Close alert which appears after Load Job gets saved successfully

14) Feel free to follow other analytic scenarios on this data. Browse R samples to get an idea of what you could do with it.
