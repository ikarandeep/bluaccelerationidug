Analyze with Cognos

The Scenario

Adele the business analyst at ACME Corporation wants to create reports using BLU Acceleration and Cognos Business Insight on her sales data. She first would like to get acquainted with the sample reports provided with BLU Acceleration for cloud and then use this scenario to create a report.

Note: Adele needs admin access to be able to perform publishing of Cube. Please ask me so that I can assign you admin privilege. You also need to download and install the following Cognos client software on your laptop. Follow Setup -> Download page. Please ignore installing fixpack and interim fixpack outlined in 2 and 3. Please install Cube designer for this scenario.

Install the IBM Cognos modeling tools on a Windows computer in the same directory in this order

  1. IBM Cognos Framework Manager (647 MB)
  2. IBM Cognos Business Intelligence 10.2.1 Fix Pack 1 (2.4 GB)
  3. IBM Cognos Business Intelligence 10.2.1.1 Interim Fix 3 (794 MB)
  4. Set up the connection to the Cognos server (Important setup steps)
  5. Optional. To model dimensional dynamic cubes: IBM Cognos Cube Designer (87.7 MB)

1) Please follow Analyze -> Cognos Business Intelligence to browse sample Cognos reports.

2) Launch IBM Cognos Cube Designer (You should have configured your “IBM Cognos Configuration” to point to the Cognos server by following step 4 in above section)

3) Click “Create New from Metadata”

4) Click OK on the “Invalid Certificate” window

5) Login with your username and password

6) Select schema “DB2INST1” from BLUDB, and click OK

7) Expand BLUDB → DB2INST1 → Tables to view the table list under schema “DB2INST1”

8) Scroll down, find table “PERSON”, right click the table and select Generate → Cube with dimensions using data sampling

9) You can rename the generated cube name as you like

10) Right click the cube and click "Pulish"

11) Select all options on the “Publish” window and click OK (You can rename your package as you like)

12) Get Deploy Results

13) Launch IBM Cognos Connection home page, select “My Folders” tab and check that the published cube is listed

14) Launch Report Studio

15) Select the published package under “My Folders”

16) Select Create new, and then click “Chart”

17) Select the “Stacked column” type

18) Pause the pointer over the query explorer button and click Queries
