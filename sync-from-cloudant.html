Sync from Cloudant

Roger the data analyst at Food product company would like to analyze USDA Food and vitamin data freely available from the web for creating new business opportunity for his food products company. He manages his NOSQL data in IBM Cloudant database and wants to leverage BLU Acceleration for Cloud for statistical analysis or reporting purposes.

In this scenario, you will be using US department of Agriculture Food database

You have a choice of downloading the JSON data from http://ashleyw.co.uk/files/foods-2011-10-03.json.zip and uploading to a Cloudant database at https://cloudant.com
OR, for the purpose of this workshop, we have published this dataset for IDUG attendees here: https://idug.cloudant.com/food

1) Login to BLUAcceleration web console. Click Manage -> Sync from Cloudant

2) Enter the following url in to Cloudant db URL

3) Enter a table name of your choice in Target Table name

4) Click Start Sync. You should see sync in progress:

5) Refresh the screen -> You should see status Running

6) You can go to Manage -> Work with Tables -> find your schema name and browse table that you just created for Cloudant sync

7) You should be able to browse data as it comes in

8) Click Manage -> Run Query -> feel free to build the query using Select Tables and Columns or type your own query.

9) You can bring the data back to excel using ODC scenario for further analytic processing. OR, follow any other scenario such as Analyze with R or Analyze with Cognos on this data.
